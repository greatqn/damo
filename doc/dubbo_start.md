# 项目启动

1.运行install.bat。成功。

2.[dubbo的测试](./doc/dubbo_test.md)

3. damo-api

java -jar damo-api-2.0.0.jar

访问 http://localhost:8080/swagger-ui.html

体验swagger，jwt的操作。

4. damo-api-test

体验 testNG 的测试用例。 testcase里的内容。

5. apllo 配置中心

只存了一些脚本，配置文件。运行文件需到官网下载。

6. nacos

新款配置中心，注册中心。但dubbo只支持到2.6.5。 可关注的一个项目。
